Hospedagem Github

Para criar ums site no github.com devemos ter uma conta válida e um repositório com um arquivo index.html

Hospedando um site

  1. Criar um repositório
  2. Publicar o repositório utilizando o publish
  3. Abrir o repositório no VScode
  4. Criar o index.html
  5. Criar a versão utilizando o commit
  6. Enviar para o repositório remoto utilizando o push
  7. Entrar no github.com, acessar o repositório ->Settings->Pages->Escolher a branch do source
  8. Clicar na URL gerada
